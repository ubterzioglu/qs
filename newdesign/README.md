# newdesign/ — "Architectural Blueprint" design backup

This is a **frozen snapshot** of the first design of the Qualtron Sinclair site
(the light drafting-paper "Architectural Blueprint" look), taken before we redesigned
the live pages to resemble the original qualtronsinclair.com look (see `ref/`).

- Snapshot date: 2026-07-05
- Contents: `app/` (all pages), `components/`, `content/`, `globals.css`
- **Do not edit or import from this folder.** It is an archive/reference only.
  The active site lives in `src/`. To restore this design, copy these back into `src/`.

Why kept: so the Blueprint version is never lost when the new (original-style)
design replaces it in `src/`.
